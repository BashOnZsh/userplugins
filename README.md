# Plugins Utilisateurs Equicord/Vencord

Ce dépôt contient des plugins utilisateurs pour Equicord et Vencord.

## Prérequis

Avant de commencer, assurez-vous d'avoir installé les outils suivants :

- **Git** (pour cloner les dépôts) : [Windows](https://gitforwindows.org/) | [macOS](https://git-scm.com/download/mac) | [Linux](https://git-scm.com/download/linux)
- **Node.js** (environnement d'exécution) : [Télécharger (Toutes plateformes)](https://nodejs.org/fr/download/)
- **pnpm** (gestionnaire de paquets) : [Instructions d'installation](https://pnpm.io/fr/installation)

## Comment les utiliser

1. Clonez ou téléchargez ce dépôt.
2. Allez dans le répertoire du code source de votre client Equicord ou Vencord.
3. Cherchez le dossier `src/userplugins`. S'il n'existe pas, créez-le.
4. Copiez les dossiers ou fichiers des plugins que vous souhaitez utiliser depuis ce dépôt vers le dossier `src/userplugins`.
5. Recompilez votre client Equicord ou Vencord (généralement en exécutant `pnpm build` ou `npm run build` selon les instructions du client).
6. Redémarrez votre client Discord pour appliquer les changements.
